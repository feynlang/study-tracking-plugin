# 문제풀이형 산출물 변형 생성 가이드

## 4가지 변형 생성 패턴

다음 난이도/볼륨 중 4개로 변형:

1. **Warmup set** — 기초 5문제, 개념 확인용 (~2-3시간)
2. **Standard set** — 중간 난이도 10문제, 응용 포함 (~1일)
3. **Challenge set** — 어려운 5문제, 깊은 이해 요구 (~1-2일)
4. **Mixed marathon** — 난이도 섞은 20문제, 종합 점검 (~2-3일)

문제 출처 후보 (토픽에 따라 적절히 제안):
- LeetCode, Codeforces, HackerRank
- 교재 챕터 끝 문제
- Project Euler (수학)
- 자체 출제 (논문 결과 재현 등)

## 완료 체크 항목 작성 가이드

각 변형의 완료 항목에 다음을 포함:
- N개 문제 목록이 명시됨 (출처/번호 포함)
- 각 문제마다 풀이 코드/노트 + 정답 확인 흔적
- 틀린 문제는 오답 노트 1줄 이상
- (선택) 본인이 막혔던 패턴 회고 1문단

## 좋은 예 / 나쁜 예

**나쁨**: "그래프 알고리즘 문제 좀 풀어본다"
**좋음**:
- [ ] 다음 10문제 풀이 완료 (LeetCode #200, #207, #210, #261, #323, #399, #547, #684, #785, #802)
- [ ] 각 문제마다 `solutions/<id>.py`로 저장
- [ ] 통과한 문제는 LeetCode 결과 스크린샷 또는 통과 시간 기록
- [ ] 틀렸거나 시간 초과한 문제는 `notes.md`에 오답 1줄 + 해결한 풀이
