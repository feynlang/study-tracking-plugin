# 링크 모드: 블로그/글 학습 산출물 변형 가이드

## 4가지 변형

1. **요약 + 비판** — 핵심 주장 정리 + 동의/반대 부분 (~반나절)
2. **응용 사례** — 글의 아이디어를 다른 도메인에 적용 (~1-2일)
3. **재현 실험** — 글의 주장을 검증하는 실험/코드 (~1-3일)
4. **반박/보완 글** — 같은 주제로 본인 관점 글 쓰기 (~1-2일)

## 완료 체크 항목 작성 가이드

### 요약 + 비판
- [ ] 글의 핵심 주장 N개를 본인 언어로 추출
- [ ] 각 주장에 대한 본인 평가 (동의 / 부분 동의 / 반대)
- [ ] 동의 시: 추가 근거 또는 사례
- [ ] 반대 시: 반박 논리 또는 반례

### 응용 사례
- [ ] 글의 핵심 아이디어 1개 명시
- [ ] 적용할 다른 도메인 1개 선정
- [ ] 적용 시뮬레이션 (사고 실험 또는 실제 코드)
- [ ] 적용이 잘 되는 부분 / 안 되는 부분 분석

### 재현 실험
- [ ] 글의 주장 중 검증 가능한 것 1개 선택
- [ ] 검증 방법 설계
- [ ] 실험 코드/데이터 작성
- [ ] 결과 분석 (글의 주장과 일치/불일치)

### 반박/보완 글
- [ ] 본인 관점의 핵심 메시지 1줄
- [ ] 글 작성 (1500~3000자)
- [ ] 원글 인용 + 본인 추가 논리
- [ ] (선택) 원글 작성자에게 코멘트 또는 이메일

## 변형 추천 가이드

글의 성격에 따라:
- 의견/주장이 강한 글 → 요약+비판 또는 반박글
- 기술적/실증적 글 → 재현 실험
- 추상적/개념적 글 → 응용 사례

수준에 따라:
- 초급 → 요약+비판 (안전한 시작)
- 중급/상급 → 응용 / 재현 / 반박
