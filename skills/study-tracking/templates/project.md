# 프로젝트형 산출물 변형 생성 가이드

목표를 받으면 다음 4가지 스코프 단계로 변형을 만든다:

1. **Tiny** — 1 모듈/1 파일, 몇 시간
2. **Small** — 작동하는 데모, 0.5~2일
3. **Medium** — 실제 사용 가능한 도구, 2~5일
4. **Stretch** — 시각화/UI/배포까지, 1주+

각 변형마다 다음 필드를 채워서 제시:
- 이름 (한 줄)
- 무엇을 만드는지 (1-2문장)
- 예상 소요 시간

## 완료 조건 작성 시 포함해야 할 항목

- [ ] 특정 파일/엔트리포인트가 존재
- [ ] 명시적 입력 → 명시적 출력이 검증됨
- [ ] README 또는 사용법 1문단
- [ ] (선택) 테스트 1개 이상 통과
