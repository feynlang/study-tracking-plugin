# 문제풀이형 산출물 변형 생성 가이드

다음 4가지 난이도/볼륨으로 변형을 만든다:

1. **Warmup set** — 기초 5문제, 개념 확인용
2. **Standard set** — 중간 난이도 10문제, 응용 포함
3. **Challenge set** — 어려운 문제 5개, 깊은 이해 요구
4. **Mixed marathon** — 난이도 섞어서 20문제, 종합 점검

문제 출처 후보: LeetCode, 교재 챕터 끝 문제, Project Euler, 자체 출제 등.
목표 영역에 따라 적절히 제안.

## 완료 조건 작성 시 포함해야 할 항목

- [ ] N개 문제 목록이 명시됨
- [ ] 각 문제마다 풀이 + 정답 확인 흔적
- [ ] 틀린 문제는 오답 노트 1줄 이상
- [ ] (선택) 본인이 막혔던 패턴 회고 1문단
