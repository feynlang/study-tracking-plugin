# 노트형 산출물 변형 생성 가이드

다음 4가지 스타일로 변형을 만든다:

1. **Cheat sheet** — 1페이지 요약, 핵심 공식/개념만
2. **Concept map** — 개념 간 관계도 + 각 개념 1문단 설명
3. **Deep notes** — 챕터별 상세 노트, 예제 + 본인 언어로 재구성
4. **Q&A 형식** — 본인이 자문자답하는 형태로 정리, 빈틈 발견용

## 완료 조건 작성 시 포함해야 할 항목

- [ ] 특정 파일이 존재 (`notes.md` 등)
- [ ] 다루기로 한 토픽 N개가 모두 포함됨 (목록을 명시)
- [ ] 본인 언어로 재구성된 흔적 (단순 복붙 금지)
- [ ] 최소 한 개의 본인 예시 또는 비유
