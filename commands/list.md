description: 진행 중인 모든 학습 목표를 진행률과 함께 표시한다
---

study-goal 스킬에서 관리하는 모든 학습 목표를 조회해서 사용자에게 보여준다.

수행할 작업:
1. `~/study-log/` 안의 모든 디렉토리 (메타 디렉토리 제외) 순회
2. 각 디렉토리의 `artifact.md`를 읽어 다음 정보 추출:
   - 토픽/콘텐츠 제목
   - 시작일
   - 모드 (일반 토픽 / 링크)
   - 상태 (🟡 진행 중 / 🟢 완료 / 🔴 일시중지)
   - 완료 체크 항목 진행률 (M/N)
   - 마지막 작업 로그 시각
3. `.active`와 비교해 어느 게 활성 목표인지 표시
4. 표 형식으로 정리해서 보여줌

진행 중 목표가 많으면 활성/비활성 / 완료 그룹으로 나눠서 표시.
