# 설명 자료형 산출물 변형 생성 가이드

다음 4가지 형식으로 변형을 만든다:

1. **블로그 포스트** — 1500~3000자, 도입-본문-마무리
2. **5분 발표 슬라이드** — 5~8장, 핵심만
3. **Feynman 설명문** — "초보자에게 5분 안에 설명" 글 한 편
4. **튜토리얼** — 코드/예제와 함께 단계별 따라하기

## 완료 조건 작성 시 포함해야 할 항목

- [ ] 산출물 파일 존재
- [ ] 가상의 청중 페르소나 명시 (예: "ML 입문 1개월차")
- [ ] 비유 또는 시각 자료 최소 1개
- [ ] (선택) 누군가에게 실제로 보여주고 피드백 받기
