# 설명 자료형 산출물 변형 생성 가이드

## 4가지 변형 생성 패턴

다음 형식 중 4개로 변형:

1. **블로그 포스트** — 1500~3000자, 도입-본문-마무리 (~1-2일)
2. **5분 발표 슬라이드** — 5~8장, 핵심만 (~반나절)
3. **Feynman 설명문** — 초보자에게 5분 안에 설명하는 글 (~반나절)
4. **튜토리얼** — 코드/예제와 함께 단계별 따라하기 (~1-2일)

## 완료 체크 항목 작성 가이드

각 변형의 완료 항목에 다음을 포함:
- 산출물 파일 존재
- **가상의 청중 페르소나 명시** (예: "ML 입문 1개월차", "비개발자 친구")
- 비유 또는 시각 자료 최소 1개
- (선택) 누군가에게 실제로 보여주고 피드백 받기
- (선택) 발표나 글 게시
