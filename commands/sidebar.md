study-goal 스킬의 샛길 질문 처리 흐름으로 진입한다.

사용자 질문: $ARGUMENTS (없으면 사용자에게 질문 받기)

규칙:
- 답변은 기본 3-5문장 핵심만
- 사용자가 "더 자세히" 1번이면 1-2문단 추가
- 2-3번째 "더"에서는 "별도 학습 목표로 만들까요?" 제안
- 답변 후 artifact.md의 "🔍 샛길 질문" 섹션에 기록:
  - 질문 / 맥락 / 핵심 답변 / 본 학습과의 관계 / 더 파볼 가치(★)
- 답변 끝에 "본 작업으로 돌아갈까요?" 자동 제안

활성 목표가 여러 개면 어느 목표의 샛길 질문인지 사용자에게 묻기.
