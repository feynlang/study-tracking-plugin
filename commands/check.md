---
description: 진행 중인 학습 목표의 완료 조건을 체크한다
---

study-goal 스킬의 완료 점검 흐름을 실행한다.

`.active`에서 활성 목표 목록을 읽고 (여러 개면 어느 것 점검할지 묻기),
선택된 목표의 `artifact.md`에서 완료 체크 항목을 가져와 사용자가 만든
산출물(`deliverable/` 안)과 비교 평가해라.

평가 결과:
- 완료: 모든 항목 만족 → 완료 플로우 (상태 변경, 회고, 루브릭 다듬기 제안)
- 거의 완료: 1-2개 누락 → 짧은 todo
- 진행 중: 다수 누락 → 다음 1-2개 액션만 제안
