# 링크 모드: 논문 학습 산출물 변형 가이드

## 4가지 변형

1. **요약 슬라이드** — 5~10장 발표 자료 (~1-2일)
2. **핵심 수식 유도** — 모든 수식 본인 손으로 다시 유도 (~1-3일)
3. **실험 재현** — 논문 결과 1개 이상 재현 (~3-7일)
4. **후속 아이디어** — 논문이 못 다룬 부분 N개 + 해결 방안 (~2-3일)

## 완료 체크 항목 작성 가이드

### 요약 슬라이드
- [ ] 5-10장 슬라이드 작성
- [ ] 각 장: 문제, 핵심 아이디어, 방법, 실험, 결과, 한계
- [ ] 본인이 만든 다이어그램 최소 1개
- [ ] (선택) 5분 발표 연습

### 핵심 수식 유도
- [ ] 논문의 모든 핵심 수식 손글씨/LaTeX로 다시 유도
- [ ] 각 단계마다 "왜 이렇게 되는가" 주석
- [ ] 막혔던 단계 표시
- [ ] 최종 결론 수식이 논문과 일치 확인

### 실험 재현
- [ ] 논문 결과 중 재현할 것 1개 선택 (작은 것부터)
- [ ] 코드 작성 (저자 코드 참고는 단계적으로)
- [ ] 데이터 준비
- [ ] 결과가 논문 ±N% 이내로 일치
- [ ] 차이점 분석 (왜 정확히 같지 않은지)

### 후속 아이디어
- [ ] 논문의 명시적 한계 N개 추출
- [ ] 논문의 암묵적 가정 N개 추출
- [ ] 각각에 대해 해결/완화 방안 제안
- [ ] 가장 유망한 1개 아이디어를 1페이지 proposal로

## 변형 추천 가이드

논문 성격에 따라:
- 이론 논문 → 핵심 수식 유도
- 실증 논문 → 실험 재현
- 응용 논문 → 후속 아이디어
- 어떤 논문이든 → 요약 슬라이드 (가벼운 시작)

수준에 따라:
- 초급 → 요약 슬라이드
- 중급 → 수식 유도 또는 후속 아이디어
- 상급 → 실험 재현 (가장 어렵고 가장 가치 있음)
